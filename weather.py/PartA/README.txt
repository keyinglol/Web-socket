[QUESTION 1 README]
---------------------------------------------------------------------------

Requirements:
1. Python 3.x
2. Flask 
Ensure the above software is installed on your computer.

---------------------------------------------------------------------------

Setup instructions:
1. Download the project folder.
2. Open in your compiler (eg. Visual Studio Code).
3. Install Flask dependencies using CLI if not already installed ( pip install Flask ).
4. In your CLI, navigate to the project folder and run ( python app.py ) to create the SQLite database.
   OR
   Alternatively, use the debugging function in your compiler.
5. Check the weather_data.sqlite using DB.Browser for SQLite.
   OR 
   build in SQLite viewer in your compiler.
   OR
   open your browser and navigate to ( http://127.0.0.1:5000/api/weather )
